## Featured Projects

### [Randomizing Groups Simulator](https://github.com/razahtet/RandomizingGroupsSimulator)
Constraint-based team allocation tool that ensures incompatible participants are never grouped unless unavoidable. Handles 200+ participants with complex constraints in under 2 seconds.

### [Shortest Path & Route Finder](https://github.com/razahtet/ShortestPathFinder)
Interactive web-based pathfinding visualizer using BFS and DFS. Allows users to create nodes, search by name, and compute shortest paths or enumerate all possible routes, with real-time canvas visualization.

### [Flood It Game](https://github.com/razahtet/floodit)
Custom implementation of the puzzle game **Flood It** with multiple board sizes, color sets, and a greedy-algorithm hint helper. Includes dynamic UI and difficulty scaling.

### [Golf Scores Prediction (In Progress)](https://github.com/razahtet/golfScores)
Sports analytics app predicting future golf scores based on historical performance and practice metrics, using linear regression. Planned features that impact predictions include weather integration and multiple ML model comparisons.
